# Quick Start Guide - Sameer Kumar Portfolio

## 🚀 Start the Project

### Option 1: Use the Start Script (Easiest)
Just double-click `start.bat` file - it will start both servers automatically!

### Option 2: Manual Start
**Terminal 1 - Backend:**
```bash
powershell -ExecutionPolicy Bypass -Command "npm run server"
```

**Terminal 2 - Frontend:**
```bash
powershell -ExecutionPolicy Bypass -Command "npm run dev"
```

## 🚢 Deploy to GitHub

```bash
# Initialize git repository
git init

# Add all files
git add .

# First commit
git commit -m "Initial commit - Portfolio website"

# Create main branch
git branch -M main

# Add your GitHub repository
git remote add origin https://github.com/YOUR-USERNAME/portfolio.git

# Push to GitHub
git push -u origin main
```

Then enable GitHub Pages in repository Settings → Pages → GitHub Actions
